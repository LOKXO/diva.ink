Moderation Commands

kick <user> [reason]

Kicks a user from the server

ban <user> [reason]

Bans a user from the server

mute <user> <duration> [reason]

Mutes a user for a specified duration in minutes

unmute <user>

Unmutes a user

clear <amount>

Clears a specified number of messages

warn <user> [reason]

Warns a user

removewarn <user> <warn_number>

Removes a specific warning from a user

warnings <user>

Views warnings for a user

timeout <user> <duration> [reason]

Timeouts a user for a specified duration in minutes

unban <user_id> [reason]

Unbans a user by their ID

addverified <user>

Adds a user to the verified list

setprefix <new_prefix>

Sets a new command prefix for the server

deleterole <role>

Deletes a role

roleinfo <role>

Displays information about a role

serverroles

Lists all roles in the server
